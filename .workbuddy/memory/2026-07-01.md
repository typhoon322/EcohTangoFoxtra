# 2026-07-01

## ETF Analysis Dashboard — Initial Build

**Stack:** FastAPI (Python) + AKShare + Sina API → Single-page HTML frontend

**Data sources (both free, no API key):**
- Real-time quotes: AKShare `fund_etf_spot_em()` → `push2delay.eastmoney.com`
- Historical K-line: Sina API `money.finance.sina.com.cn/quotes_service/api/json_v2.php/CN_MarketData.getKLineData`

**Note:** `push2his.eastmoney.com` is blocked in sandbox; Sina API works as alternative for kline data.

**Features:**
- 15 ETFs in pool (宽基/行业/跨境/商品)
- 3 strategies: momentum rotation (3M+6M), MA crossover (MA20/MA60), DCA enhancement
- 5-tab frontend: overview, rotation, trend, DCA, interactive K-line chart
- Dark theme, Chinese market color convention (红涨绿跌)

**Key files:**
- `backend/data_fetcher.py` — dual-source data layer
- `backend/signals.py` — 3 strategies signal computation
- `backend/main.py` — FastAPI server (port 8765)
- `frontend/index.html` — single-page dashboard
- `run.py` — entry point
